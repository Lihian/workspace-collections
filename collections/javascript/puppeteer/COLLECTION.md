## 搜集原因

Puppeteer 是 Google 维护的 Headless Chrome/Chromium 自动化库，也支持 Firefox。提供 JavaScript API 控制浏览器进行自动化测试、网页截图、PDF 生成、爬虫等。是浏览器自动化领域的标杆项目。
